# Interview Views and Controls

The core Watchword API that drives the interview engine is based on generators that yield React Components. This storybook provides navigable demos of the low level elements used to to compose the yielded values.

- Views
  - [Minifig](http://localhost:6006/?path=/docs/interview-minifig--docs) - Lego Minifigs showing Roles evidenced in interview.
  - [Passage](http://localhost:6006/?path=/docs/interview-passage--docs) - Passages of text rendered on screen
- Controls
  - [Tell](http://localhost:6006/?path=/docs/interview-tell--docs) - Call to action button to navigate to next passage
  - [Prompt](http://localhost:6006/?path=/docs/interview-prompt--docs) - Choice buttons to navigate to different passages
- Layouts
  - [Page](http://localhost:6006/?path=/docs/interview-minifig--docs) - embeds Minifig and Passage views alongside Tell and Prompt controls
